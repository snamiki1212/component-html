## Overview

Example WebComponent feature.

- Split html file
- Reuseble component
- Scoped CSS

## Usage

```zsh
$ yarn install
$ yarn http-server
$ open http://127.0.0.1:8080/
```

## Why Use `http-server`

Because of security feature in browser, we cannot open .js file via .html.
